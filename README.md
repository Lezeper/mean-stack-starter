# MEAN Stack Init

### RESTful API
                GET    /api/user     - find all user
                GET    /api/:id      - find specific user
                POST   /api/login    - user login
                POST   /api/user     - user register
                PUT    /api/user     - update user
                DELETE /api/user     - delete user
                
### Framework and Plugin			
				Mongoose
				Express
				AngularJS
				NodeJS
				Jwt
